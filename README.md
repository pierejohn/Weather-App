# Weather App

A responsive weather application that allows users to search for any city worldwide and view real-time weather conditions and forecasts. The application fetches live weather data from a weather API and displays detailed information including temperature, weather conditions, wind speed, and precipitation probability. Weather APIs commonly provide temperature, weather conditions, wind speed, humidity, and precipitation forecasts, which are the types of data this project displays.

## Features

* Search for weather information by city name
* Display current temperature
* Show weather conditions (Sunny, Cloudy, Rainy, Windy, etc.)
* Display wind speed information
* Show precipitation (rain) probability
* Responsive design for desktop, tablet, and mobile devices
* Dynamic weather icons based on current conditions
* Real-time weather updates using API data

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap
* Weather API
* REST API
* Fetch API

## How It Works

1. Enter the name of any city.
2. Submit the search request.
3. The application sends a request to the Weather API.
4. Weather data is retrieved and displayed on the page.
5. Users can view:

   * Current temperature
   * Weather condition
   * Wind speed
   * Chance of rain
   * Additional forecast information

## Installation
